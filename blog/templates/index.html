what is a client?

A user or the users machine that obtains information by making requests from the server.

what is a server?

a computer/machine that holds information and databases that accepts requests from client machines and responds with the info they are requesting.

what is an HTTP request?

An HTTP request is the combination of the URL (that represents the resource the client is attempting to access), the request type (GET, CREATE, PUT/POST, or DELETE), the meta data, and sometimes a body of data.

what is an HTTP response?

An HTTP Response is what is returned from the HTTP request. This would be the Status code (404 Not Found if it doesn't exist, 200 means a-okay!), the Header(meta data), and the body, which is the data that the user interacts with.

what is Django and why do we use it?

Django is a framework for Python that helps us build web apps and connect them to databases. It takes care of a lot of hassle involved in the web development; enables users to focus on developing components needed for their application.
